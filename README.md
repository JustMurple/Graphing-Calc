Axis

The project I made consists in four interactive math tools to help you study and understand more advanced math topics such as multi-variable functions, derivatives, intgrals, vector fields and much more!

Tools:

1) Single variable graphing calculator. This takes one or more functions of x as an input and shows the user their graph. It also comes with 4 analysis tools: computing the derivatives, the arclength and average of a                                               function between two integers a and b, and showing the rational zeros of set functions.

2) 3D graphing calculator. This works similarly to the single variable but it takes functions of multiple variables as inputs. It takes as input functions z= f(x,y) and shows the 3D plot of them. The 3D grapher come with                               the partial derivatives and critical points tools, which respectively take the partial derivatives of the input functions and show their critical points, along with some extra information.

3) Symbolic calculator. This comes with a multitude of different mathematical operations such as iterated integrals, derivatives, limits and antiderivatives. It lets you customize everything about the function and even                             plots the single variable ones (those used in expressions, definite integrals, integrals, derivatives and limits).

4) Vector Field plotter. This last tool is more useful for those studying vector calculus as not only it plots 2D vector fields but it can also take their curl, divergence and work/flux integrals given x,y parametrization                           in t.
